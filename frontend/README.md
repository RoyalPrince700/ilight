# I-LIGHT Solar Website (Frontend)

Modern multi-page company website built with React + Vite for **I-LIGHT SOLAR AND ELECTRICAL ENERGY NIG. LTD**.

## Tech Stack

- React
- Vite
- React Router
- TypeScript

## Pages Included

- Home
- About
- Projects
- Products
- Services
- Contact

## Design Direction

The UI follows a premium, minimal, modern aesthetic inspired by high-end product brands (clean layout, strong typography, soft gradients, and polished card surfaces) while keeping content easy to read and conversion-friendly.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview production build:

```bash
npm run preview
```
